## 1. Project Overview & Scenario
This portfolio contains a comprehensive, hands-on implementation of an Information Security Management System (ISMS) for a fictional enterprise. It simulates the exact responsibilities required of a GRC Engineer: translating technical IT infrastructure states into risk metrics, managing frameworks, and preparing documentation for internal/external audits.

### Enterprise Profile: "Cyberdome GmbH"
* **Industry:** Cloud-based Software-as-a-Service (SaaS) Platform Provider
* **Location:** Leipzig, Germany (Subject to EU GDPR regulations)
* **Infrastructure:** Hybrid cloud environment (AWS Cloud infrastructure paired with hybrid on-premise Active Directory domain controllers)
* **Target Objective:** Establish an asset risk register, construct cross-functional technical security policies, map overlapping controls across frameworks, and prepare the organization for a stage-1 ISO 27001 certification audit.
